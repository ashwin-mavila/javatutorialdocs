Collections SynchronizedMap

Synchronized Map synchronizes map operations making HashMaps thread-safe.

Its performance is not as good as ConcurrentHashMap. Hence it is recommended to use ConcurrentHashMap instead of SynchronizedMap or HashTable.

Sample Program:

            import java.util.Collections;
            import java.util.HashMap;
            import java.util.Map;

            public class SynchronizedMapDemo {
                public static void main(String[] args) {
                    HashMap map = new HashMap<>();
            
                    Map syncMap = Collections.synchronizedMap(map);
            
                    syncMap.put(1, "One");
                    syncMap.put(2, "Two");
                    syncMap.put(3, "Three");
            
                    System.out.println("map[1]: " + syncMap.get(1));
                    System.out.println("map: " + syncMap);
                }
            }

Output:

            map[1]: One
            map: {1=One, 2=Two, 3=Three}

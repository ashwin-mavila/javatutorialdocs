Collections Frequency

Frequency method returns number of occurrances of objects in a collection.

Sample Program:

            import java.util.ArrayList;
            import java.util.Collections;
            import java.util.List;

            public class FrequencyDemo {
                public static void main(String[] args) {
                    List mylist = new ArrayList();
                    mylist.add("alpha");
                    mylist.add("beta");
                    mylist.add("cupcake");
                    mylist.add("jellybean");
                    mylist.add("jellybean");
                    mylist.add("jellybean");
                    mylist.add("kitkat");
                    mylist.add("jellybean");

                    String toFind1 = "kitkat";
                    int freq1 = Collections.frequency(mylist, toFind1);

                    String toFind2 = "pie";
                    int freq2 = Collections.frequency(mylist, toFind2);

                    System.out.println("freq of " + toFind1 + ": " + freq1);
                    System.out.println("freq of " + toFind2 + ": " + freq2);
                }
            }

Output:

            freq of kitkat: 1
            freq of pie: 0

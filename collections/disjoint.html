Collections Disjoint

Disjoint method returns true if two given collections have no elements in common, else false.

Syntax:

            public static boolean disjoint(Collection c1, Collection c2)

Throws

  1. NullPointerException: If either collection is null.
  2. NullPointerException: If one collection contains a null element and null is not an eligible element for the other collection.
  3. ClassCastException: If one collection contains an element that is of a type which is ineligible for the other collection.

Sample Program:

            import java.util.ArrayList;
            import java.util.Collections;
            import java.util.List;

            public class DisjointDemo {
                public static void main(String[] args) {
                    List list1 = new ArrayList<>();
                    list1.add("one");
                    list1.add("two");
                    list1.add("three");

                    List list2 = new ArrayList<>();
                    list2.add("ten");
                    list2.add("twenty");
                    list2.add("thirty");

                    System.out.println("disjoint lists: " + Collections.disjoint(list1, list2));
                }
            }

Output:

            disjoint lists: true
